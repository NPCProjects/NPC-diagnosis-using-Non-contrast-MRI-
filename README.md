
# README

## Overview

This repository contains Python scripts for processing, training, and evaluating a deep learning model designed for multi-modal medical image analysis. The code focuses on leveraging MRI data for diagnosing nasopharyngeal carcinoma (NPC). The project includes various components such as data preprocessing, model training and evaluation.

#### Usage
Adjust the data path as needed

Traing a new model
```bash
python train.py
```

Testing a model
```bash
python test.py
```
